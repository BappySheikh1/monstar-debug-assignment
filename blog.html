প্রশ্ন-১ sessionStorage এবং LocalStorage এর মধ্য পার্থক্য কী?

উত্তরঃ session storage-কোনো একটা website এর storage! কোনো একটা জিনিস কে store করার জন্য একটা জাইগা সেখানে যে জিনিস টা রাখি সেটা temporary ! যতক্ষণ browser চালউ থাকবে ততক্ষণ পর্যন্ত থাকে ! যদি browser টাকে close করে দেয় তাহলে session Storage এর জিনিস গুলো নাই হয়ে যাবে !

প্রশ্ন-২ Block scope এবং Global scope এর মধ্য পার্থক্য কী?

প্রশ্ন-৩ JavaScript এর Event Loop কিভাবে কাজ করে?

উত্তরঃ Event loop (ইভেন্ট লুপ) জাভাস্ক্রিপ্টের একটি সিক্রেট machenism যার মাধ্যমে জাভাস্ক্রিপ্ট single-threaded প্রোগ্রামিং ল্যাঙ্গুয়েজ হওয়া সত্ত্বেও বাহ্যিক ভাবে multi-threaded প্রোগ্রামিং ল্যাঙ্গুয়েজের মত কাজ করে। ইভেন্ট লুপ গভীরভাবে call stack কে পর্যবেক্ষণ করে এবং যদি call stack খালি বা empty থাকে তবে Event queue থেকে Task call stack এ পাঠায় execution সম্পন্ন করার জন্য। আমরা সকলে জানি, জাভাস্ক্রিপ্ট একটি সিঙ্গেল-থ্রেডেড asynchronous প্রোগ্রামিং ল্যাঙ্গুয়েজ । এইটা বিষয় লক্ষ্য করেছেন কি, একটা ল্যাঙ্গুয়েজ কি করে একই সাথে সিঙ্গেল-থ্রেডেড আবার asynchronous হতে পারে? আসলে বিষয়টা হল, জাভাস্ক্রিপ্ট একটা সিঙ্গেল-থ্রেডেড ল্যাঙ্গুয়েজ; তার মানে হল জাভাস্ক্রিপ্ট একসাথে একই সময়ে একটা মাত্র কাজ করতে পারে। আর asynchronous বিষয়টা জাভাস্ক্রিপ্ট ল্যাঙ্গুয়েজের কোন বিষয় না, এটি নিয়ন্ত্রিত হয় ব্রাউজার Enviornment এর মাধ্যমে।

type of Undefiend -8 ways
1.variable that is not initialized will give Undefiend ! 2.function with no return ! 3.parameter that is not passed will be Undefiend ! 4.if return has nothing on the right side will return Undefiend ! 5.কোনো একটা property by default ভাবে থাকে object এর ভিতর না থাকে তাহলে Undefiend দেখায় ! 6. একটা array এর ভিতর যদি value এর বেশি index দিয়ে খোজা হয় তাহলে Undefiend দেখায় ! 7.একটা Array Element Delete করলে output Undefiend দেখায় ! 8. ডিরেক্ট একটা variable এর সাথে Undefiend সেট করলে output Undefiend দেখায় !
